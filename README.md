# YOLO Support Page

Простая страница поддержки для App Store Connect.

## Как загрузить на GitHub

1. Создайте новый публичный репозиторий на GitHub (например, `yolo-support`)
2. Загрузите файл `index.html` в корень репозитория
3. Включите GitHub Pages в настройках репозитория
4. Получите URL: `https://ваш-username.github.io/yolo-support/`
